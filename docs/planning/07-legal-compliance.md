# 07. 법적 고려사항 & 컴플라이언스

> ⚠️ 본 문서는 법률 자문이 아닙니다. 실제 서비스 런칭 전 반드시 변호사/금감원 공식 가이드를 확인해야 합니다.

## 7.1 핵심 리스크: 유사투자자문업

### 근거 법령
- **자본시장과 금융투자업에 관한 법률** (자본시장법) 제101조
- 유사투자자문업: 불특정 다수인을 대상으로 간행물·출판물·통신물·방송 등을 통해 투자조언 제공
- **금감원 신고 대상** (등록이 아닌 "신고")

### 우리 서비스의 해석
"매수/매도 타이밍 제공"은 **그 자체로 유사투자자문업 소지가 매우 높음.**

하지만 아래 조건을 충족하면 **"정보 제공 서비스"** 로 볼 여지 있음:
1. 개인 맞춤형이 아님 (사용자 포트폴리오/재무상태를 반영하지 않음)
2. 특정 종목에 대한 "매수/매도 권유"가 아닌 **기술적 지표와 뉴스의 객관적 요약**
3. "투자 조언이 아닌 참고 정보임" 명시
4. 대가(유료 구독료) 없음 — 유료화 시점에 재검토 필요

→ **A안 채택: "분석 도구" 포지셔닝**.

## 7.2 A안 실천 규칙

### 용어 사용 (서비스 전체에 일관 적용)

**❌ 금지 용어**
- "매수 추천", "매도 추천"
- "사세요", "파세요", "들어가세요"
- "이 종목은 오릅니다/내립니다"
- "투자 조언", "재무 상담"
- "수익 보장", "손실 없음"

**✅ 권장 용어**
- "기술적 시그널", "분석 지표"
- "강세/약세 신호", "중립"
- "과매수/과매도 구간"
- "정보 제공", "참고 자료"
- "투자 판단 보조"

### 면책 고지 (Disclaimer)

**모든 페이지 하단 + AI 분석 카드 내부**에 고정 표시:

```
본 서비스에서 제공되는 모든 정보는 투자 참고용이며,
투자 자문이나 매수/매도 권유가 아닙니다.
모든 투자 판단과 그에 따른 책임은 사용자 본인에게 있습니다.
과거 성과는 미래 수익을 보장하지 않습니다.
```

**AI 분석 결과 카드**에는 추가로:

```
본 AI 분석은 공개된 시장 데이터와 뉴스를 기반으로 한
알고리즘 출력이며, 전문 금융 자문이 아닙니다.
```

### 프롬프트 수준의 제약

LLM 프롬프트에 강제:
- "투자 권유 표현을 사용하지 말 것"
- "가격 예측 수치를 생성하지 말 것"
- "확정적 어조 대신 가능성/경향 어조 사용"
- summary_ko 마지막에 면책 문구 포함

## 7.4 외부 API 및 데이터 재배포

### Finnhub
- 무료 플랜: 개인/비상업적 용도 허용. 상업 재배포 제한.
- 서비스 내에서 **가공/표시**는 허용, 원시 데이터 **재판매 금지**.
- 유료화 단계에서 약관 재확인 필수.

### Yahoo Finance
- 공식 API 없음. 비공식 이용은 **TOS 회색지대**.
- 상업 서비스에선 fallback 용도로만 제한 사용 권장.

### TradingView Lightweight Charts
- Apache 2.0 라이선스 — **상업적 사용 가능**.
- Advanced Charts는 별도 신청/심사 필요.

### 뉴스 기사
- 원문 **전문 재게재 금지** (저작권)
- 제목 + 요약 + 원문 링크는 **공정이용(fair use)** 소지 있으나 판례마다 다름
- 안전한 방법: **LLM이 만든 요약문 + 원문 링크** (요약은 2차 저작물이므로 상대적으로 안전)

## 7.7 유료화 시점의 재검토 포인트

**무료 단계와 유료 단계는 법적 위험이 다릅니다.**

유료 구독을 도입하면:
- 자본시장법상 "대가" 요건 충족 → 유사투자자문업 신고 가능성 대폭 상승
- 이 경우 **금감원 유사투자자문업 신고** 진행 (개인사업자 기준 약 1~2주 소요, 자본금 요건 없음)
- 또는 순수한 "정보 제공 서비스" (뉴스 큐레이션 수준)로 포지셔닝 조정

→ **수익화는 반드시 법적 검토 후 결정.**
