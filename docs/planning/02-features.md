# 02. 기능 명세

## 2.1 기능 맵

```
┌─ 메인(시장 대시보드) ──────────────────┐
│  · 지수(S&P500, Nasdaq, Dow) 스냅샷     │
│  · VIX, USD/KRW, 미국 10Y 금리          │
│  · 시장 뉴스 피드 (한국어 요약)         │
│  · 인기 종목 / 급등락 종목              │
└──────────────────────────────────────┘
           ↓ 티커 입력/검색
┌─ 종목 상세 페이지 ─────────────────────┐
│  · 기본 정보 (회사명, 섹터, 시총 등)    │
│  · 실시간(15분 지연) 가격 + 변화율       │
│  · 차트 (1D/1W/1M/3M/1Y, 지표 토글)    │
│  · 기술적 지표 카드 (MACD/볼밴/RSI 등)  │
│  · AI 분석 시그널 + 근거                │
│  · 종목 뉴스 피드 (한국어 요약)         │
│  · [북마크] 버튼                        │
└──────────────────────────────────────┘
┌─ 마이페이지 ──────────────────────────┐
│  · 북마크 목록 (간단 뷰)               │
│  · 알림 설정 (종목별 on/off, 조건)     │
└──────────────────────────────────────┘
```

## 2.2 기능 상세

### F1. 종목 검색 & 기본 정보

- **입력:** 티커(`AAPL`) 또는 회사명(`애플`, `Apple`)
- **자동완성:** Finnhub `/search` API
- **표시 정보:** 회사명(영문/한글), 섹터, 산업, 시가총액, 직원수, 상장 거래소, 공식 홈페이지 링크

### F2. 차트 (TradingView 스타일)

- **라이브러리:** TradingView Lightweight Charts
- **타임프레임:** 1일/1주/1개월/3개월/1년/5년
- **표시 요소:**
  - 캔들스틱
  - 이동평균선 (MA5, MA20, MA60)
  - 볼린저밴드 오버레이 (토글)
  - 거래량 바 (하단)
- **MVP에서 제외:** 드로잉 툴, 지지선/저항선, 실시간 스트리밍

### F3. 기술적 지표 카드

현재값 + 간단한 **초보용 해설 툴팁**을 함께 표시.

| 지표 | 표시 형식 | 해설 (툴팁) 예시 |
|---|---|---|
| RSI (14) | 숫자 + 게이지 | "70 이상은 과매수, 30 이하는 과매도 신호" |
| MACD | 값 + 시그널선 + 히스토그램 | "0선 돌파 시 추세 전환 가능성" |
| 볼린저밴드 | 상단/중앙/하단 값 + 현재 위치 | "밴드 상단 터치는 단기 고점 신호일 수 있음" |
| 이동평균선 | MA5/20/60 현재값 | "단기선이 장기선을 돌파하면 상승 신호(골든크로스)" |
| 거래량 변화 | 20일 평균 대비 % | "평소보다 거래량 급증은 주목 신호" |

→ 서버(Spring Boot)에서 `ta4j` 라이브러리로 계산.

### F4. AI 분석 시그널 ⭐ 핵심 기능

**입력 컨텍스트 (서버에서 수집):**
- 기술적 지표 수치 (F3 전체)
- 최근 7일 가격 액션 요약 (고가/저가/변동성)
- 해당 종목 최신 뉴스 5~10건 (헤드라인 + 요약)
- 시장 컨텍스트 (VIX, 해당 섹터 움직임)

**출력 (LLM JSON 응답):**
```json
{
  "signal": "weak_buy",
  "confidence": 0.68,
  "timeframe": "단기(1~2주)",
  "rationale": [
    "RSI 45로 중립, MACD 히스토그램 양전환",
    "최근 실적 호조 뉴스 다수",
    "시장 전반 변동성(VIX 18)은 안정적"
  ],
  "risks": [
    "다음 주 FOMC 회의로 변동성 확대 가능",
    "볼린저밴드 상단 근접으로 단기 조정 가능"
  ],
  "summary_ko": "전반적으로 긍정적 신호..."
}
```

**캐싱:** 동일 종목 1시간 캐시 → 비용 절감 + 일관성.

→ 상세는 [05-ai-strategy.md](05-ai-strategy.md) 참고.

### F5. 뉴스 피드 (종목 / 시장)

- **소스:** Finnhub company-news, market-news
- **LLM 번역/요약:** 원문 영어 → 한국어 2~3줄 요약 (Gemini Flash)
- **캐싱:** 기사 단위 24시간 캐시 (같은 기사 재번역 방지)
- **표시:**
  - 제목(한글 번역)
  - 요약(한글 2~3줄)
  - 출처, 발행 시각
  - 원문 링크

### F6. 시장 대시보드 (메인)

| 섹션 | 데이터 | 소스 |
|---|---|---|
| 주요 지수 | S&P500, Nasdaq, Dow, Russell 2000 | Finnhub / Yahoo |
| 공포 지수 | VIX | Finnhub / Yahoo |
| 환율 | USD/KRW | Finnhub / exchangerate API |
| 금리 | 미국 10Y 국채 금리 | Yahoo |
| 시장 뉴스 | 상위 10건 | Finnhub market-news |
| 인기/급등락 | Top gainers/losers | Finnhub |

### F7. 북마크

- 로그인 사용자 기능
- 종목별 [북마크] 토글
- 마이페이지에서 목록 + 간단 스냅샷

### F8. 푸시 알림

**알림 조건 (사용자 설정):**
- 가격 변동 임계치 (예: ±5%)
- 새로운 주요 뉴스 발생 시
- AI 시그널 변화 시 (예: 중립 → 매수)

**구현:**
- MVP: Web Push API (PWA, 무료)
- 향후 앱: Firebase Cloud Messaging

**주기:** Spring Boot 스케줄러가 5~15분 간격으로 북마크 종목 체크.

## 2.3 비기능 요구사항

| 항목 | 목표 |
|---|---|
| 페이지 로딩 | P95 < 2s |
| AI 응답 | 캐시 hit 시 < 200ms, miss 시 < 5s |
| 가용성 | 99% (무료 티어 한도 내 최선) |
| 모바일 대응 | 반응형 (MVP는 데스크톱 우선, 모바일 깨지지 않게만) |
| 접근성 | 키보드 네비게이션, 색약 고려 |

## 2.4 MVP에서 제외 (의식적 축소)

- 포트폴리오 관리
- 백테스팅
- 실시간 스트리밍 시세
- 소셜 기능 (댓글, 공유)
- 개인화 추천
- 수익률 시뮬레이션

→ 전부 Phase 5+ 로 후순위. MVP는 **"1개 종목을 잘 보여주는 것"** 에 집중.
