# 01. 서비스 개요 & 포지셔닝

## 1.1 서비스 한 줄 정의

> **"AI가 차트·뉴스·시장 지표를 종합해 초보 투자자에게 친절한 참고 시그널을 제공하는 미국 주식 분석 도구"**

## 1.2 문제 정의

초보 투자자가 주식을 볼 때 겪는 어려움:

- 차트만 봐도 뭐가 뭔지 모름 (MACD? 볼린저밴드? RSI?)
- 뉴스는 영어라서 읽기 부담
- 종목 뉴스 따로, 시장 뉴스 따로, VIX 따로 — **흩어져 있음**
- "지금이 살 타이밍인가?"에 대한 **직관적 판단 도구가 없음**
- 유튜브/블로그는 신뢰도가 들쭉날쭉

## 1.3 타겟 사용자

**페르소나:** 미국 주식에 관심은 있지만 지표/차트/영어 뉴스에 익숙하지 않은 **초보 투자자**

- 증권 앱은 있지만 "그래서 뭘 사야 하지?"에서 막힘
- 하루 10~30분 정도 투자 정보를 보는 라이트 유저
- 데스크톱 기준으로 여유롭게 보고, 알림은 모바일로 받고 싶어함

## 1.4 서비스 포지셔닝 (A안: 분석 도구)

**유사투자자문업 신고를 피하는 "분석 도구" 포지셔닝**을 채택.

- AI 출력은 **"매수/매도 추천"이 아닌 "시그널/점수/근거 표시"** 형태
  - 예: `강한 매수 / 약한 매수 / 중립 / 약한 매도 / 강한 매도` + 신뢰도 %
- 모든 페이지에 **면책 고지** 컴포넌트 노출
- 특정 종목을 "사라"고 적극 권유하지 않음
- 사용자 개인별 맞춤 자문은 제공하지 않음 (일반화된 분석만)

상세 법적 가이드는 [07-legal-compliance.md](07-legal-compliance.md) 참고.

## 1.5 핵심 차별점

| 기존 서비스 | AI Stock Advisor |
|---|---|
| Yahoo Finance — 정보 많지만 영어, 분산 | **한국어로 통합된 대시보드** |
| 인베스팅닷컴 — UI 복잡, 초보에 부담 | **초보 친화 UI + 지표 해설** |
| 증권사 HTS/MTS — 데이터만 있고 해석 없음 | **AI 해석/시그널 제공** |
| 유튜브 종목 분석 — 일방적, 편향 | **실시간 데이터 기반 객관적 분석** |

## 1.6 제약 조건

- **비용:** 월 운영비 $15 이하 목표 (MVP 기준)
- **실시간성:** 15분 지연 시세 허용 (유료 API 미사용)
- **법적:** 유사투자자문업 신고 없이 운영 (A안 준수)
- **언어:** 한국어 UI, 영어 원문 뉴스는 LLM 번역/요약

## 1.7 수익 모델

**현재 단계: 완전 무료, 수익화 고려하지 않음.**

향후 사용자가 쌓이면 검토할 수 있는 옵션:
- 프리미엄 구독 (더 많은 북마크, 알림 빈도 상향, 고급 지표)
- 제휴 마케팅 (증권사 계좌 개설)
- 광고 (배너/네이티브)

→ 수익화는 Phase 5 이후 논의.
